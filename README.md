# 激光雕刻图像预处理器 (img4laser)

这是一个用于优化激光雕刻图像的Web应用工具。该工具可以自动分析图像特征，并根据不同类型的图像（普通照片、卡通/线稿、人像）应用最佳的处理策略。

## 主要功能

- 自动检测图像类型（普通照片、卡通/线稿、人像）
- 基于图像特征优化锚点灰度值
- 智能抖动处理
- 实时预览处理结果
- 直方图显示
- 详细的图像分析报告

## 技术特点

- 纯前端JavaScript实现
- 模块化设计
- 自适应处理算法
- 无需后端服务器

## 使用方法

1. 上传图像
2. 系统自动分析并处理图像
3. 查看处理结果和分析报告
4. 可选手动调整参数
5. 下载处理后的图像
